# To-Do App (Backend + Frontend + Docker)

A full-stack To-Do application featuring:
- Backend: Node.js + Express, JWT authentication, MongoDB persistence
- Frontend: React + Vite, API consumption via Axios
- Environment orchestration with Docker Compose

## Tech Stack

- Backend
  - Node.js
  - Express 5
  - Mongoose (MongoDB)
  - JSON Web Token (JWT)
  - CORS
- Frontend
  - React 18
  - Vite
  - Axios
- Dev tooling
  - Nodemon (backend dev)
  - ESLint (frontend)
  - Docker Compose
